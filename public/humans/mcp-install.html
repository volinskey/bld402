Get started

Copy one of these into your coding agent (Claude Code, Cursor, or similar). Your agent handles the install and starts building.

Install + build in one go

Replace the last part with whatever you want to build:

Install the bld402-mcp plugin, then build me a shared todo app

That's it. Your agent will install the plugin, pick a template, set up the database, deploy the app, and give you a shareable link.

More examples

Try any of these — or describe your own idea:

Install the bld402-mcp plugin, then build me a voting app where I create polls and friends vote by visiting a link
Install the bld402-mcp plugin, then build me a trivia game where I write questions and friends join with a code
Install the bld402-mcp plugin, then build me a landing page where visitors can sign up for my product waitlist

Already have the plugin installed?

Then just ask your agent to build something. No need to say "install" again:

Use bld402 to build me a recipe sharing app where people can post and browse recipes

What happens next?

  1. Your agent installs the plugin (takes a few seconds)
  2. It picks a template or builds from scratch based on your description
  3. It sets up everything — database, security, hosting
  4. You get a link like my-app.run402.com that works on any device
  5. Want changes? Just tell your agent — updates are free
