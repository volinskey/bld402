Install bld402-mcp

One config change. Then ask your AI assistant to build something.

Claude Code

Add this to your Claude Code MCP config (~/.claude/claude_desktop_config.json or project .mcp.json):

{
  "mcpServers": {
    "bld402": {
      "command": "npx",
      "args": ["bld402-mcp"]
    }
  }
}

Then ask Claude: "What templates does bld402 have?" or "Build me a shared todo app."

Cursor

Open Cursor Settings → MCP Servers → Add new server:

{
  "mcpServers": {
    "bld402": {
      "command": "npx",
      "args": ["bld402-mcp"]
    }
  }
}

Then ask Cursor: "Build me a hangman game with bld402."

Other MCP-compatible tools

Any tool that supports Model Context Protocol can use bld402-mcp. The config is the same — point to npx bld402-mcp as the command.

Command-line (no AI assistant)

You can also use the bld402 CLI directly:

# Install globally
npm install -g bld402-mcp

# Browse templates
bld402 browse list

# Build an app
bld402 build --name my-app --template shared-todo

# Check status
bld402 status

# Remove when done
bld402 remove

What happens next?

  1. Your assistant discovers 5 tools: browse, build, update, status, remove
  2. Ask it to build something — it picks a template or builds from scratch
  3. It handles everything: wallet, payments, database, security, hosting
  4. You get a live URL like my-app.run402.com
  5. Ask for changes anytime — updates are free

Requirements

  • Node.js 18+ — needed to run the MCP server
  • Internet connection — to reach the run402 API
  • No sign-up, no API keys, no credit card needed
