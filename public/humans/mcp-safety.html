MCP Safety

What bld402-mcp can and cannot do.

What it CAN do

  • Create web apps on run402 servers
  • Set up databases, user accounts, and file storage
  • Deploy serverless functions
  • Deploy static websites with custom subdomains
  • Delete apps you've created

What it CANNOT do

  • Access your local files or filesystem
  • Send emails, SMS, or push notifications
  • Access other websites or APIs on your behalf
  • Install software on your computer
  • Read your browser history or cookies
  • Access anything outside of run402 infrastructure

Cost limits

bld402-mcp uses testnet money (Base Sepolia USDC) by default. This is free — it is not real money. There is no risk of unexpected charges.

Tier Cost Duration What you get
Prototype $0.10 (testnet) 7 days Database, hosting, 5 functions, 250 MB storage
Hobby $5.00 30 days Database, hosting, 25 functions, 1 GB storage
Team $20.00 30 days Database, hosting, 100 functions, 10 GB storage

The prototype tier uses testnet USDC from a free faucet. You will never be charged real money unless you explicitly upgrade to Hobby or Team.

Data handling

  • Wallet key is stored locally at ~/.config/run402/wallet.json with restricted file permissions
  • Session data (project ID, API keys) is stored locally at ~/.config/run402/bld402-session.json
  • App data (database contents, uploaded files) lives on run402 servers (AWS, US region)
  • No telemetry — bld402-mcp does not send analytics or usage data anywhere
  • No third-party services — all communication is between your machine and api.run402.com

Open source

bld402-mcp is open source under the MIT license. You can inspect every line of code on GitHub.
