What is MCP?

Give your AI assistant the power to build and deploy web apps.

MCP (Model Context Protocol) is a standard that lets AI assistants use tools. Think of it as giving your assistant hands — instead of just talking about code, it can actually build and deploy things.

bld402-mcp is a set of tools that plug into any MCP-compatible assistant. Once installed, your assistant can build complete web apps, deploy them to a live URL, and iterate on them — all from a single conversation.

How it works

You describe what you want

"Build me a trivia game where friends can join with a code."

↓

Your AI assistant uses bld402-mcp

It picks a template (or builds from scratch), sets up the database, configures security, and deploys the site.

↓

bld402-mcp talks to run402

All the infrastructure — database, hosting, security, payments — is handled by run402.com.

↓

You get a live app

A shareable link like trivia.run402.com that works on any device. Built in minutes.

What tools does it include?

Tool What it does
bld402_browse Browse 13 ready-made templates, get source code, or read the build guide
bld402_build Build and deploy a complete web app in one call — returns a live URL
bld402_update Update a deployed app: change the UI, add database columns, update functions
bld402_status Check what's deployed: project, database tables, live URL
bld402_remove Delete the app and clean up when you're done

Which AI assistants support it?

  • Claude Code — full support
  • Cursor — full support
  • ChatGPT Codex — MCP support varies
  • Any tool that supports Model Context Protocol

You can also use the bld402 command-line tool directly, without an AI assistant.
